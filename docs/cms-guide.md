# 海得宝官网 · 内容管理后台 使用说明

给运营人员的操作指南。后台用于管理官网的「新闻」和「产品」内容。

---

## 一、登录前的准备（只需做一次）

1. 你需要一个 **GitHub 账号**（github.com 免费注册）。
2. 管理员会用 GitHub 把你**邀请为协作者**。请检查你 GitHub 注册邮箱里的邀请邮件，点邮件里的 **「Accept invitation」**（接受邀请）。
   - ⚠️ 邀请 **7 天内有效**，请尽快接受。
   - 只有接受邀请后，你保存的修改才能真正生效。

## 二、登录后台

1. 浏览器打开：**https://www.martempo.com.cn/admin/**
2. 点 **「使用 GitHub 登录」**
3. 首次登录会跳转到 GitHub 授权页，点 **Authorize（授权）**
4. 授权后自动进入后台

## 三、可以管理的内容

- **新闻** —— 发布品牌新闻、动态
- **产品** —— 维护产品信息（名称、副标题、图片、规格、成分、详情、购买链接等）

## 四、怎么编辑

1. 左侧选 **「新闻」** 或 **「产品」**
2. **新建**：点右上角 **「New 新闻 / New 产品」**；**修改**：点列表里已有的条目
3. 按表单填写或修改（每个字段下方有灰色说明提示）；图片字段直接点选本地文件上传
4. 完成后点右上角 **「Publish 发布」→「Publish now 立即发布」**

## 五、发布后多久生效

- 点「立即发布」后，网站会**自动重新生成**，大约 **1～2 分钟**后线上更新
- 刷新对应页面即可看到改动

## 六、注意事项

- ✅ 一定要点 **「Publish now 立即发布」** 才会上线；只点 Save 是草稿，不会生效
- 🖼️ 上传图片前最好先压缩（图片太大影响网站打开速度），建议单张控制在 500KB 以内
- ⚠️ 所有改动会**直接影响线上官网**，删除/修改请谨慎确认
- 🔒 每个人请用**自己的** GitHub 账号登录（系统会记录是谁改的，方便追溯）
- ❓ 如果保存时报错（例如提示无权限 / 403），多半是仓库邀请还没接受，请联系管理员
